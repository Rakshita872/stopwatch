## 📌 Project Overview

This project helps users:
- Track elapsed time using a Stopwatch
- Set and run a countdown Timer
- Pratice DOM manipulation and JavaScript timing functions

## 🚀 Features

#⏰ Stopwatch
- Start timing
- Stop/pause timing
- Rest Timer

# ⏳ Countdown Timer
- Set minutes and seconds
- Start countdown
- Reset Timer
